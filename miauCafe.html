MiauCafé

Sitio web responsivo, con base de datos para un café universitario chileno. Este proyecto fue realizado con una mezcla de Design Thinking y SCRUM, que fue registrado en Jira.

Fui encargada de la elaboración de HTML, CSS y Javascript de la sección de menú, inicio de sesión y pedidos; y elaboración de la base de datos. Adicionalmente, realicé las pantallas de alta fidelidad acorde al diseño atómico acordado y los wireframes provistos
